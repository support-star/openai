# Programmieren Skill

Dieses Skill-Set erweitert Claude um erweiterte Programmier-Fähigkeiten über den Code-Generator hinaus.

## Fähigkeiten

### 1. Code-Analyse (`/code-analyse`)
- Bestehenden Code analysieren und dokumentieren
- Komplexität messen (zyklomatische Komplexität)
- Doppelten Code finden und refactoren
- Performance-Engpässe identifizieren
- Verbesserungsvorschläge mit Codebeispielen

### 2. API-Entwicklung (`/api <methode> <endpunkt>`)
- REST-API-Endpunkte implementieren
- Request/Response-Formate definieren (JSON)
- Fehlerbehandlung und HTTP-Statuscodes
- API-Dokumentation generieren (OpenAPI/Swagger)
- Mock-Server für Frontend-Entwicklung

### 3. Frontend-Entwicklung (`/frontend <typ>`)
- Responsive Layouts erstellen (CSS Grid, Flexbox)
- Interaktive UI-Komponenten (Modals, Tabs, Akkordeons)
- Formulare mit Echtzeit-Validierung
- Drag & Drop Funktionalität
- Animationen und Übergänge (CSS/JS)

### 4. Datenvisualisierung (`/chart <typ>`)
- Diagramme erstellen (Balken, Linien, Kreis, Gauge)
- SVG-basierte Charts ohne externe Bibliotheken
- Echtzeit-Datenaktualisierung
- Interaktive Tooltips und Legenden
- Druckoptimierte Darstellung

### 5. Testing (`/unittest <datei>`)
- Unit-Tests schreiben (Vanilla JS)
- Integrationstests für UI-Komponenten
- Test-Runner im Browser implementieren
- Testabdeckung messen
- Automatisierte Regressionstests

### 6. Debugging (`/debug`)
- Fehler analysieren und beheben
- Performance-Profiling
- Memory-Leak-Erkennung
- Browser-Kompatibilität prüfen
- Console-Logging-Strategie implementieren

## Regeln
- Vanilla JS bevorzugen (kein Framework)
- ES6+ Syntax verwenden
- Sauberer, lesbarer Code (Clean Code Prinzipien)
- Fehlerbehandlung bei allen externen Aufrufen
- Kommentare nur wo nötig, Code spricht für sich
